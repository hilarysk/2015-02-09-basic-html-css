How do they work?

In a nutshell, forms get input from the user and send it to either a website or a web server. They can collect selections from a pre-determined list, or let the user input anything they want, or let the user input data according to specific criteria. You see these everywhere - any time you create an account for a website, login to a site, or fill out a SurveyMonkey survey for you friend's cousin's grad school research on the sexual health of biracial double-amputee lesbians who drive 1994 Honda Civics and live in California cities that begin with "L".

How do you format them?

We'll take a look at how to code three formats: selecting an option from a drop-down menu, clicking a "radio button" (think round checkbox) or entering freeform data.

First, freeform data:

_________________

First name:

Last name:



_________________

The code appears thusly:

<form action="">
<input type="text" name="firstname" value="ie: Flora">
<br>
Last name:<br>
<input type="text" name="lastname" value="ie: Zeng">
<br><br>
<br><br>
<input type="submit" value="Submit">
</form>

Pretty simple. If this were a working form, pressing "Submit" would send the information to wherever the "action" attribute in line 1 points (in this case, nowhere).
