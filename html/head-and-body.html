Head

The head is so meta.

No, but seriously. The <head> section is where you store the page's metadata, or data about data. For example, this page's head includes the following:

<head>
<title>HTML and CSS | OCSWiki</title>
<meta name="head and body" content="Wiki project about HTML and CSS to practice HTML and CSS. This is meta meta-content."/>
<link rel="CSS goes here" href=""/>
<head>

The <link rel="CSS goes here" href=""/> is where we would reference our CSS, if we had one for the site (and which I'm pretty sure I have to make soon).

Body

The body is where the content lives. All of this, from Head on, is part of the body. It's where the guts of the page lives.
