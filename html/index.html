HTML

HTML is the language of the internet. (AP Style still dictates that "internet" be capitalized - at least, last time I checked - but I think that's a silly convention and I will continue to ignore it. Moving on.) HTML stands for "hypertext markup language," and it's basically the public face of a program. The underlying program might be written in any number of languages, but it can only interact with your web browser through HTML. It's like the melting-pot of programming.

Or, you know, imperialist colonizers forcing assimilation.

Either way, in this section, we'll be exploring the following:

  • Forms
  • Headers and paragraphs
  • Images
  • Lists
