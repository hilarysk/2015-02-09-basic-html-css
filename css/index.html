CSS

CSS makes things beautiful. Elegant. FABULOUS.

Basically, CSS (or Cascading Style Sheets) contains all the formatting and none of the content for a website. You can make specific paragraphs red, add a background image, italicize all your links or even - god forbid - make the text for your site something horrendous like ComicSans.

I can't stress this enough: with great power comes great responsibility.

This section includes some common CSS vocabulary, as well as information about different ways to incorporate CSS into your content.
