CSS > Inline, Embedding and External Styles

There are three different styles of including ... style.

Inline styles

This is when code is written directly into the tag for the element, as we demonstrate on the images page. This method only affects the tag in which it's included. Here's another example:

<a href="http://reddit.com" style="text-decoration: none;">

Embedded styles

This is formatting code is embedded in a <style> tag within the <head> of the page. This only affects tags on that page. Example:

<style type="text/css">
p {
color: #00f;
}
</style>

External styles

This refers to the use of an external document in which the formatting is coded; that document (or "stylehseet") is then attached to HTML pages. Example:

<link rel="stylesheet" type="text/css" href="styles.css">
