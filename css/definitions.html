CSS > Definitions

  • CSS selector: Represents a section of code for which you can write formatting rules in your stylesheet.

  • Type selector: Corresponds with an HTML element type, such as <body> or <head>

  • ID selector: When called with "#", references a specific section of your code.

  • <div id="word">: You can use this to assign unique ID selectors to different sections of your code.

  • Property: Similar to an attribute in Ruby. It's a different aspect of formatting, set to a value. Ie:

    body {
    color: red

    }
    Where "color" is the property and "red" is the value.

  • Padding: The space between the content and the edge of an element

  • Margin: The space around an element

  • Rule set: Refers to a selector and its curly braces, properties and values

  • Declaration block: The code between the curly braces (includes the declarations, or property-value pairs)
